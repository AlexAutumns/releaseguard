# ReleaseGuard

ReleaseGuard is a gamified software release-risk investigation simulator for beginner software developers.

## MVP Scope

The MVP is a desktop-first local web game where players inspect fictional release tickets, review evidence cards, file structured findings, submit release verdicts, and receive scoring reports.

## Tech Stack

- React Router Framework Mode
- React
- TypeScript
- Vite
- Tailwind CSS
- Local browser storage

## Current Status

Milestone 2 planning is complete. Development is starting with the first playable release ticket vertical slice.

## Development Goal

Build one complete gameplay loop before expanding content or visual polish.

## Planned Core Loop

1. Start a shift
2. Open a release ticket
3. Inspect evidence cards
4. Pin or select clues
5. File structured findings
6. Submit a release verdict
7. View ticket report
8. Save local progress
